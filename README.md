### Portfolio Website ###

Static portfolio site for **Evelin Eilmess**.

## Live site
https://evelineilmess.github.io

## Project structure
- `index.html` – homepage
- `globals.css` – resets + font loading
- `styleguide.css` – design tokens (colors, typography variables)
- `style.css` – page/component styles
- `image/` – exported images and SVGs
- `assets/font/` – local fonts (Berghan + Work Sans)

## How to edit
- Update text and links in `index.html`
- Replace project images in `image/` (keep filenames the same, or update paths in HTML)
- Add new project pages in `projects/<project-name>/index.html`

## Deployment
Hosted on **GitHub Pages** (Settings → Pages → Deploy from `main` / root).
